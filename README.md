# List to List – Android Task & Notes Manager

## 📱 Deskripsi Aplikasi
**List to List** adalah aplikasi Android yang digunakan untuk mencatat dan mengelola berbagai aktivitas harian seperti tugas, pekerjaan, daftar belanja, dan catatan pribadi.  
Aplikasi ini membantu pengguna mengatur kegiatan berdasarkan prioritas dan waktu secara terstruktur, sehingga aktivitas lebih terorganisir dan mudah dipantau.

---

## ✨ Fitur Utama

### 📝 Manajemen Catatan dan Tugas
- Menambahkan catatan atau tugas baru
- Mengedit judul, isi, tanggal, dan waktu tugas
- Menghapus tugas dengan konfirmasi (tidak langsung terhapus)
- Memperbarui catatan kapan saja

### 🎯 Kategori Prioritas
Tugas dikategorikan berdasarkan tingkat prioritas dengan warna berbeda:
- 🟢 **High** (tinggi)
- 🟡 **Medium** (sedang)
- 🔴 **Low** (rendah)

### ⏰ Pengelompokan Berdasarkan Waktu
- Tugas diatur berdasarkan tanggal dan waktu
- Memudahkan penjadwalan kegiatan
- Menampilkan urutan tugas secara kronologis

### 📅 Tampilan Kalender
- Melihat tugas pada tanggal tertentu
- Menampilkan tugas hari ini dan sebelumnya
- Menampilkan tugas yang sudah lewat

### 🗂️ Riwayat (History)
- Tugas yang terhapus atau terlewat masuk ke riwayat
- Tugas dapat dipulihkan kembali dari riwayat
- Data tidak langsung hilang permanen

### 👤 Profil Pengguna
- Mengedit data profil pengguna

### 🔒 Keamanan Penghapusan
- Tugas tidak langsung terhapus saat tombol delete ditekan
- Sistem konfirmasi mencegah kesalahan pengguna

## 🛠️ Teknologi yang Digunakan
- Android Studio
- Java
- XML Layout
- SQLite / Local Storage
